Discord RPG Bot: A Text-Based Adventure with Evolving Dragons

I built an immersive, turn-based combat Discord bot in Java using Spring Boot and JDA (Java Discord API), designed to bring RPG-style mechanics into community servers. Inspired by classic role-playing games, the bot allows users to hatch, raise, and battle with dragons in a persistent game world.

Key Features:

  • Dragon Evolution System: Users receive a dragon egg, which evolves over time into a dragonling and eventually a mighty dragon. Evolution is tracked using timestamps and JSON-based persistence.
  • Turn-Based Combat: Players can initiate combat against dynamically generated enemies, take turns attacking, and gain experience based on victory.
  • Real-Time Battle Threads: Each battle occurs in its own Discord thread for clarity and immersion, showing real-time HP updates, turn management, and attack outcomes.
  • Experience and Progression: Dragons gain XP from combat, allowing for stat progression and future scalability for leveling, abilities, or forms.
  • Combat Engine: Built using modular services and utility classes for attack logic, timers, and status tracking, with a 1-second turn loop using ScheduledExecutorService.

Tech Stack:

  • Java + Spring Boot for scalable backend structure
  • JDA (Java Discord API) for Discord integration
  • JSON file-based persistence for user and dragon state tracking
  • Scheduled task system to handle timed evolution and combat turns

This project demonstrates my ability to blend object-oriented design, asynchronous task management, and game mechanics into a fully interactive, community-driven bot. It’s a living platform I plan to expand with features like PvP battles, loot systems, and elemental abilities.
